Заголовок: Корректный запрос
Описание: Проверка того, что бэк принимает запрос, пытается записать данные в бд и дает ответ что все прошло успешно
Шаги:
1) Запусаем, например, Burp Suite
Результат - должен быть запущен Burp Suite
2) вводим http запрос на бэк сайта fintech.tinkoff.ru
Результат - мы должны видеть форму для тестов
3) Вводим значения для полей апи
Результат - верные поля
4) делаем запрос
Результат - Бэк должен пытаться записать данные в бд (ну скорее всего в бд) и должен сообщить нам об этом в своем ответе
